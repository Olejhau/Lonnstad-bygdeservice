Bilder til nettsiden
Nettsiden henter bilder automatisk fra disse to mappene i GitHub:
```txt
bilder/hero
bilder/galleri
```
Hero-bilder
Legg bilder til toppen av nettsiden i:
```txt
bilder/hero
```
Nettsiden viser automatisk de 3 nyeste bildene i hero-seksjonen.
Bildekarusell
Legg bilder til bildekarusellen i:
```txt
bilder/galleri
```
Nettsiden viser automatisk de 15 nyeste bildene i karusellen.
Støttede filtyper
```txt
.jpg
.jpeg
.png
.webp
.gif
.svg
```
Filnavn
Filnavn spiller ingen rolle. Du kan for eksempel bruke:
```txt
jobb-1.jpg
transport.png
graving-fåberg.webp
```
Tips
Bruk helst liggende bilder i god kvalitet. Hero-bilder bør være brede, gjerne minst 1600 px brede.
